simple css editor for wp-login page

== Description ==
twp login is a simple to use css editor for the wp-admin/wp-login.php page of any wordpress website.
It allows to change the page colors and logo, also its possible to redirect this page to another one, hidding this way the wp-admin and wp-login pages.
Easy, fast and simple, customize your website login page in seconds.
